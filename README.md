# Music downloader

A utility that fetches all tracks in a YouTube Music playlist and tries to match and download them from Tidal.

It also supports a Plex integration to prevent duplicate downloads.

## Setup
1. Install docker with compose
2. Get the [compose file](docker-compose.yml)
3. Run `docker compose up -d`

The app is available at port 3000
